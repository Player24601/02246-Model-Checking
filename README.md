# 02246 Model Checking

This repository contains course materials for the Model Checking course (02246).

## Course Structure

This repository is organized into the following sections:

- **lectures/** - Lecture slides and notes
- **assignments/** - Course assignments and projects
- **exercises/** - Practice exercises and solutions
- **examples/** - Example models and verification scenarios
- **resources/** - Additional reading materials and references

## About Model Checking

Model checking is a formal verification technique used to verify finite-state concurrent systems. This course covers the theoretical foundations and practical applications of model checking.

Browse the folders above to access course materials. Each folder contains a README with specific information about its contents.
